BidLens UI Style Recommendations

基于 BidLens 的产品定位 —— 招标文档语义比对工具，面向采购/工程专业人员，以下是三种推荐的 UI 风格方案

Style A Recommended

Swiss Minimalism / 瑞士极简主义

以内容为中心，去除一切装饰噪音。网格系统 + 高对比度排版，让评审人员专注于文档差异本身。

Why this style fits BidLens:
  • 降低认知负荷 — 招标文档通常非常密集（表格、条款、清单），极简界面减少视觉噪音，让用户专注于差异内容
  • 专业可信 — 瑞士风格源自国际主义设计，天然传递「精确、可靠、专业」的气质，符合采购/工程用户的审美预期
  • 高对比度排版 — 纯白底 + 深色文字 + 语义化高亮色（黄/绿/红），差异一目了然
  • 可读性最佳 — Inter + Noto Sans SC 字体组合，中英文混排清晰，长时间审阅不疲劳
  • WCAG AAA 无障碍 — 最高可达无障碍等级，适合企业级合规要求

Style B

Data-Dense Dashboard / 数据密集仪表盘

深蓝侧边栏 + 浅灰内容区 + 统计卡片，最大化信息密度。适合需要快速浏览大量差异项、进行批量评审的场景。

Why this style fits BidLens:
  • 信息密度最大化 — 统计卡片 + 紧凑表格 + 过滤器，让用户一眼掌握全局差异分布
  • 蓝色信任体系 — 深蓝 #1e40af 传递「数据可靠、分析专业」的企业级气质
  • 结构化数据展示 — 与 BidLens 的 DiffSummary（modified/added/deleted/moved 等分类）天然契合
  • 批量评审效率 — 紧凑的表格行高 + 筛选器 + 进度条，适合处理 50+ 差异项的大文档比对
  • 专业工具感 — 类似 Jira / Linear / Azure DevOps 的布局范式，工程/采购用户熟悉且信赖

Style C

Dark Professional / 暗色专业模式

深色背景 + 靛蓝强调色，减少长时间审阅的眼部疲劳。差异高亮在暗色底上更加醒目，适合加班/夜间评审场景。

Why this style fits BidLens:
  • 护眼舒适 — 招标文档审阅往往需要数小时，暗色背景显著降低眼部疲劳
  • 差异高亮更醒目 — 黄/绿/红高亮在深色底上对比度更高，差异一目了然
  • 现代专业感 — 类似 VS Code / Linear / Raycast 的暗色主题，符合技术人员审美
  • 会议/投影友好 — 在暗光会议室中，暗色界面不会「白屏刺眼」
  • 靛蓝 #4f46e5 — 既保持专业感，又比纯蓝更有品牌辨识度

Summary

综合推荐

建议采用 Style A (Swiss Minimalism) 作为默认主题，Style C (Dark Professional) 作为可选暗色模式。
  • 默认浅色 — 招标文档审阅通常在办公环境进行，浅色主题是安全选择
  • 暗色可选 — 提供暗色模式切换，满足长时间审阅和夜间工作的需求
  • Style B 的数据密集思路可以融合 — 在 Style A 的基础上，将统计卡片和筛选器整合到顶部工具栏
  • 字体统一使用 Inter + Noto Sans SC — 中英文混排最佳方案
  • 差异高亮色统一 — 修改=#fef3c7(黄) / 新增=#d1fae5(绿) / 删除=#fee2e2(红)，两套主题保持一致的语义色
