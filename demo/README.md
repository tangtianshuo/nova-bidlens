# BidLens 演示说明

## 快速开始

### 方法1：使用批处理文件（推荐）
1. 双击运行 `demo/start-server.bat`
2. 浏览器将自动打开演示页面
3. 如果没有自动打开，请访问 http://localhost:8000

### 方法2：手动启动服务器
1. 打开命令提示符或PowerShell
2. 进入项目目录：`cd D:\Projects\Nova\nova-bidlens\.worktrees\semantic-compare-mvp`
3. 运行命令：`python demo/server.py`
4. 在浏览器中访问 http://localhost:8000

## 演示功能

### 1. 三栏复核工作台
- **左侧**：差异导航面板，显示所有识别的差异项
- **中间**：文档对比区域，左右并排显示两个文档
- **右侧**：差异详情面板，显示选中差异的详细信息

### 2. 交互功能
- 点击左侧差异项，查看对应的文档内容
- 文档中的差异内容会高亮显示
- 右侧显示差异的置信度、相似度和摘要信息

### 3. 核心特性展示
- 智能文档解析
- 语义比对引擎
- 差异智能分析
- 报告自动生成
