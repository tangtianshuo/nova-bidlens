🚀 核心功能特性

📄 智能文档解析

支持Word文档(.docx)的智能解析，自动提取文本内容并构建结构化文档AST，为后续比对提供准确的数据基础。

🔍 语义比对引擎

基于Rust高性能引擎，实现文档的语义级比对，不仅识别文字差异，更能理解文档语义变化。

📊 差异智能分析

自动识别新增、删除、修改、移动等多种差异类型，提供置信度和相似度评分，帮助用户快速定位关键变化。

🖥️ 三栏复核界面

创新的三栏布局设计，左侧差异导航、中间文档对比、右侧详情面板，提供直观高效的复核体验。

📝 报告自动生成

支持Markdown和HTML格式的比对报告自动生成，包含完整的差异统计和详细分析，便于存档和分享。

⚡ 高性能架构

采用Electron + React + Rust技术栈，前端响应迅速，后端处理高效，支持大文档的快速比对。
