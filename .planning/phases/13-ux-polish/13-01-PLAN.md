**UX-05: Concurrency limiter for MinerU requests**

1. Add a simple concurrency limiter at the top of parser-service.ts:
```typescript
// Concurrency limiter for MinerU cloud API (max 2 concurrent)
let mineruInFlight = 0;
const MINERU_MAX_CONCURRENT = 2;
const mineruQueue: Array<() => void> = [];

function acquireMinerUSlot(): Promise<void> {
  if (mineruInFlight < MINERU_MAX_CONCURRENT) {
    mineruInFlight++;
    return Promise.resolve();
  }
  return new Promise(resolve => { mineruQueue.push(resolve); });
}

function releaseMinerUSlot(): void {
  const next = mineruQueue.shift();
  if (next) {
    next(); // hand off slot directly
  } else {
    mineruInFlight--;
  }
}
```

2. Add offline detection before MinerU parse in `parseDocumentFile`:
```typescript
// Check network before attempting MinerU cloud parse
const { net } = require('electron');
if (!net.isOnline()) {
  return {
    success: false, warnings: [], duration: 0, parserId: 'mineru-parser',
    error: { code: 'MINERU_OFFLINE', message: '此文件需要云端解析，请检查网络连接' },
  };
}
```
Wait, `net.isOnline()` is not a real API. Use a simpler approach: catch fetch errors and check for network errors. Or use `dns.lookup` to check connectivity. Actually the simplest approach: the MinerU parser already catches network errors via `withRetry`. The `fetch failed` pattern already catches offline. We just need to detect it and map to MINERU_OFFLINE.

Better approach: add a quick connectivity check before calling MinerU:
```typescript
async function isOnline(): Promise<boolean> {
  try {
    const { lookup } = await import('node:dns/promises');
    await lookup('mineru.net');
    return true;
  } catch {
    return false;
  }
}
```

3. Wrap MinerU calls with concurrency limiter and offline check in `parseDocumentFile`:
Where we call `mineru.parse(input, options)`, wrap it:
```typescript
// Check offline
if (!(await isOnline())) {
  return {
    success: false, warnings: [], duration: 0, parserId: 'mineru-parser',
    error: { code: 'MINERU_OFFLINE', message: '此文件需要云端解析，请检查网络连接' },
  };
}

// Acquire concurrency slot
await acquireMinerUSlot();
try {
  const result = await mineru.parse(input, options);
  // UX-02: Reset cached parser on 401
  if (!result.success && result.error?.code === 'AUTH_EXPIRED') {
    resetMinerUParser();
  }
  return result;
} finally {
  releaseMinerUSlot();
}
```

This pattern applies at all 3 places where `mineru.parse()` is called (scanned PDF direct, pdf-parse fallback, and the non-PDF path won't hit MinerU).
