# Milestone v0.3.6: PDF 原文定位与数据提取

**Status:** SHIPPED 2026-07-24
**Phases:** 14-17
**Total Plans:** 5

## Overview

实现点击关键审查点的提取文本后，弹出 PDF 阅读器并定位到对应原文位置。支持 bbox 高亮、zoom-to-fit、tooltip 交互、双栏对比。

**Goal**: 数据管道保留 bbox 位置信息，用户可在应用内查看 PDF 文件
**Depends on**: Phase 13 (v0.3.4)
**Plans**: 2 plans

Plans:
- [x] 14-01: Data layer — types, mapper bbox, IPC endpoint
- [x] 14-02: PDF viewer — react-pdf, Drawer, nav, zoom

### Phase 15: Evidence → PDF 定位连线

**Goal**: 审查员点击 evidence 卡片即可打开对应 PDF 文件并定位到原文页面
**Depends on**: Phase 14
**Plans**: 1 plan

Plans:
- [x] 15-01: Wire evidence page badges to open PDF Drawer at correct file and page

### Phase 16: 高亮与交互增强

**Goal**: PDF 页面上高亮显示 evidence 匹配区域，支持交互查看详细信息

## Milestone Summary

**Key Accomplishments:**
1. MinerU bbox preservation through DocumentAst pipeline + risk:getPdfFile IPC
2. react-pdf canvas-based PDF viewer with page nav, zoom, scroll tracking, keyboard shortcuts
3. Evidence card page badges wired to open PDF Drawer at correct file and page
4. HighlightOverlay canvas rendering with bbox rectangles, zoom-to-fit, and tooltip
5. DualPdfDrawer with side-by-side panes and auto-detecting cross-file evidence

**Key Decisions:**
- Digital PDF bbox extraction deferred to future milestone
- Only new dependency: pdfjs-dist (react-pdf already indirect dep)
- Canvas overlay + transparent div for mouse events (not shadcn Tooltip)
- Bottom Sheet (h-[90vh]) for dual-pane with CSS grid-cols-2

**Requirements:** 17/17 complete (PDF-01 through PDF-17)
